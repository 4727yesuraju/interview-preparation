# 📚 What are the Features of JavaScript?

## 📖 Simple English Explanation

JavaScript is a powerful and flexible programming language used to build interactive web applications.

Its features make it easy to write, understand, and run on different platforms like browsers, servers (Node.js), mobile apps, and desktop applications.

---

## 🤔 Why is it Needed?

- Makes web pages interactive.
- Easy to learn and write.
- Runs on almost every browser.
- Can be used for both frontend and backend development.

## 🎤 Interview Answer (30 Seconds)

JavaScript has many powerful features. It is a **high-level**, **interpreted**, and **dynamically typed** programming language. It supports **object-oriented programming**, **event-driven programming**, and **asynchronous programming**. It is **cross-platform**, **lightweight**, and can be used for both **frontend** and **backend** development using Node.js.
